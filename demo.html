Timer Demo

Use the Timer Factory to create your own instance of timer for general purpose in your code, using your own callback function.
.create(tickIntervalMinutes, events, timoutMinutes, callback) - 4 required parameters must be provided
   where tickIntervalMinutes is numeric; events is String of events, delimited by a space (i.e. " ") to interrupt; timeoutMinutes is numeric; callback is a function. timer such that elapsed time is reset back to 0.

For example, this code snippet,
   var timer = TimerFactory.create(1/60, \"click keydown focus scroll\", 1/60*15, function(){ console.log('OK'); alert('OK'); });
would instantiate a timer with the following 'configurations':
  • tick every second
  • timer is interrupted if any of these events is encountered: mouse click, keyboard pressed, zoom in/out or page scrolling
  • timeout after 15 seconds
  • after timeout, the callback function would log "OK" in console and alert a pop-up window with "OK".
